    Простой консольный калькулятор     
    version 1.0

Это мой первый проект, нацелен на развитие умений Git/Github.
Периодически буду обновлять и дополнять по мере изучения новых технологий.

    Порядок работы: 

1. Ввести первое число. Число должно быть целым, иначе программа потребует повторного ввода. 
2. Ввести команду (+,-,*,/). Если команда введена неверно, то программа потребует повторный ввод команды.
3. Ввести второе число. Если число не целое или введено не число, программа потребует повторного ввода.
4. Вывод результата.


- Если использован "0" и команда "/", программа запросит повторный ввод команды.
- Если использован "0" во втором числе и команда "/", программа выдаст ошибку "Деление на 0 невозможно" и завершит работу.
